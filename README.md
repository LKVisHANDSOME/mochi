# Mochi

## 短期目標（分配工作後兩個禮拜內）
- [x] 使用者
    - [x] 登入
    - [x] 登出
    - [x] 修改自己的個人資料
    - 忘記密碼先省略
- [x] 題目
    - [x] 新增
        - 題目敘述為純文字
        - 新增題目需要正確程式碼、測資、限時、限記憶體
        - 新增前確認測資輸出
        - 上傳題目時必須提供題解
    - [ ] 刪減
    - [ ] 編輯
- [x] submission
    - [x] 提交 submission 並得到結果，結果存入資料庫
    - [x] 觀看全部人的 submission status
    - [x] 每個題目有各自的 submission status
    - [x] 查看使用者自己的 submission status
